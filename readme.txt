IIIF previewing and viewer display using Gutenberg blocks in WordPress. 

== Description ==

= Features =
* Insert IIIF Manifest thumbnails as blocks using Yith IIIF
* Mirador viewer displays in a modal on user action
* Gutenberg block wrapper for Yith IIIF modes
